### ∀x∀y∀z R(x, y,z)
**"Для любой тройки объектов (x, y, z) из универсума D предикат R(x, y, z) является истинным".**
- Каждая ячейка может быть заполнена независимо двумя способами: `1` (истина) или `0` (ложь).
- Общее количество таких ячеек (троек) равно `n * n * n = n³`.
- Следовательно, **общее число всех возможных интерпретаций предиката `R`** (и, соответственно, всех интерпретаций для нашей формулы) равно:  
		**`2^(n³)`**


**"Не существует ни одной тройки объектов (x, y, z) из универсума, для которой R(x, y, z) было бы ложно"**
- Мы обязаны в каждую из `n³` ячеек таблицы поставить `1`. Любое отклонение (постановка `0` в любую ячейку) сразу нарушает условие `∀x ∀y ∀z R(x, y, z)` и делает интерпретацию **не моделью**.
**Следовательно, число моделей равно 1.**


### ∀x ∀y ∃z R(x, y, z)

**"Для любой пары объектов (x, y) в универсуме существует хотя бы один объект z, такой, что тройка (x, y, z) удовлетворяет отношению R."**

Общее число интерпретаций предиката `R` по-прежнему равно `2^(n³)`.

**"В таблице предиката R нет ни одной строки (x, y), которая была бы полностью нулевой."**
Проще посчитать, сколько интерпретаций **НЕ** являются моделями. Это те интерпретации, где **хотя бы для одной пары (x, y) все ячейки по z равны 0**.
- **Сколько всего существует разных векторов длины `n`,** где каждый элемент может быть `0` или `1`?  
    По правилу произведения: для каждой из `n` позиций есть 2 варианта.  
    **Ответ: `2^n`.**
- **Сколько среди них векторов, которые состоят из одних нулей?**  
    Такой вектор всего **один**: `(0, 0, 0, ..., 0)`.
- **Сколько существует векторов, в которых есть хотя бы одна `1`?**  
    Это все векторы, кроме одного-единственного "нулевого".  
    **Ответ: `2^n - 1`.**

Формула `∀x ∀y ∃z R(x, y, z)` требует, чтобы условие `∃z R(x, y, z)` выполнялось **для каждой пары `(x, y)` одновременно**.
(2^n - 1)^(n²)


### ∃x ∀y ∀z R(x, y, z)
**"Существует такой особый объект x, что для любой пары объектов (y, z) тройка (x, y, z) удовлетворяет отношению R."**

1. **Выбор "особого" x:** Существует `n` кандидатов на роль этого особого объекта `x₀`.
2. **Требование к строке:** Для выбранного `x₀` формула требует, чтобы `R(x₀, y, z)` было истинно **для всех** `y` и `z`. Это значит, что вся "срез" таблицы (плоскость), соответствующая `x = x₀`, должна быть заполнена единицами (`1`).
    - Размер этого среза: `n * n = n²` ячеек.
    - Существует **ровно 1 способ** заполнить этот срез так, чтобы везде были единицы.
3. **Заполнение остальной таблицы:** На все остальные ячейки таблицы (где `x ≠ x₀`) **не накладывается никаких ограничений**. Их можно заполнять любым способом (`0` или `1`).
    - Количество таких ячеек: `Общее число ячеек - ячейки в срезе x₀ = n³ - n²`.
    - Количество способов заполнить их: `2^(n³ - n²)`.
4. **Комбинаторный расчёт:** Для каждого из `n` кандидатов на роль `x₀` количество подходящих таблиц равно `1 * 2^(n³ - n²)`.

### `∀x ∃y ∀z R(x, y, z)`

**"Для каждого объекта x существует свой подобранный объект y, такой, что для любого объекта z тройка (x, y, z) удовлетворяет отношению R."**

1. **Выбор помощников:** Для каждого `x` мы должны выбрать хотя бы одного `y` (помощника), который будет работать. Фактически, мы должны для каждого `x` назначить **непустое** множество `Y_x` возможных помощников. Однако условие `∀z` требует, что если `y` выбран, то **весь столбец `R(x, y, z)` для всех `z` должен состоять из единиц**. Это очень сильное требование на помощника.
2. **Требование к ячейкам:** Если для данного `x` объект `y` является помощником, то это накладывает условие на `n` ячеек: `R(x, y, z) = 1` для всех `z`.
