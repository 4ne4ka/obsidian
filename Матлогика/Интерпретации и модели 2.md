### `∃x (P(x) ∨ Q(x))`
**"В универсуме существует хотя бы один объект x, для которого истинно P(x) или истинно Q(x)."**
Другими словами: **Множества `P` и `Q` не могут быть одновременно пусты.** Хотя бы один объект должен попасть в объединение `P ∪ Q`.

1. **Общее число интерпретаций:**
    - Предикат `P` — это подмножество `D`. Число вариантов: `2ⁿ`.
    - Предикат `Q` — это подмножество `D`. Число вариантов: `2ⁿ`.
    - Поскольку `P` и `Q` интерпретируются **независимо**, общее число пар `(P, Q)` равно `2ⁿ * 2ⁿ = 2²ⁿ`.
2. **Считаем контрмодели (когда формула ложна):**  
    Формула ложна, когда **не существует** ни одного `x`, для которого `P(x) ∨ Q(x)` истинно. Это значит, что `P(x) ∨ Q(x)` ложно для **всех** `x`.
    - `P(x) ∨ Q(x)` ложно тогда и только тогда, когда ложно и `P(x)`, и `Q(x)`. То есть, когда `x ∉ P` и `x ∉ Q`.
    - Чтобы это условие выполнялось для **всех** `x` в универсуме, ни один объект не должен принадлежать ни `P`, ни `Q`.
    - Это означает, что **`P` должно быть пустым множеством** и **`Q` должно быть пустым множеством**.
3. **Число контрмоделей:** Существует **ровно 1** такая пара предикатов: `(P = ∅, Q = ∅)`.
4.  `Число_Моделей = 2²ⁿ - 1`

### `∀x (P(x) ∨ Q(x))`
**"Для любого объекта x в универсуме истинно P(x) или истинно Q(x)."**
Другими словами: **Каждый элемент универсума должен принадлежать хотя бы одному из множеств P или Q.** То есть, объединение `P ∪ Q` должно покрывать весь универсум `D`.
1. **Общее число интерпретаций:** Как и раньше, `2²ⁿ`.
    
2. **Анализ условия для одного объекта:**  
    Рассмотрим один фиксированный объект `a ∈ D`. Какие пары `(P, Q)` удовлетворяют условию `P(a) ∨ Q(a)`?
    - Всего возможных пар значений `(P(a), Q(a))`: 4.
        - `(0, 0)` → **Запрещено** (условие ложно).
        - `(0, 1)` → Разрешено.
        - `(1, 0)` → Разрешено.
        - `(1, 1)` → Разрешено.
3. **Число моделей:**  
    Для каждого из `n` объектов в универсуме мы независимо выбираем одну из `3` разрешённых пар значений.  
    `Число_Моделей = 3 * 3 * ... * 3 = 3ⁿ`
